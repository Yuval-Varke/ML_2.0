kNN - k Nearest Neighbours
    - majorly used for classification problems, can also be used for Regression problems.
    - Always scale data for kNN problems
1. Choosing k (hyperparameter) / Always odd
2. Distance of new point from all the k data points
3. If the problem is classification = majority of k Neighbours is taken
   Else if the problem is Regression = mean/avg of k Neighbours is taken
To calculate Distance :- 
a. Euclidean distance (mostly used)
b. Manhattan distance (generally used for grid based problems/ high dimensional data)

#Limitation:-
-> Slow at predictions (not suitable for huge datasets)
-> Sensitive to outliers
-> Sensitive to feature scaling
-> Curse of Dimensionality (High Dimensional Data) - Data Sparsity

Cross Validation (used to get the best values for the hyperparameters):-
-> K-fold CV --- gets k accuracy values for k iterations, at the end mean of all those accuracy score is 
                finalized.
             --- if k = cv = 5, then the train dataset will be divided into 5 folds and each iteration/fold
             is used as a Validation data, we'll be getting diff acc_scores.
