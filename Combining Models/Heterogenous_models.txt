Uptill now we knew of these supervised machine learning models like linear regression, 
logistic regression, DTs, SVMs,etc.
In the ensemble learning techniques of Bagging and Boosting, multiple models were used,
which were same, thats why they were known as Homogenous Ensemble models.

Heterogenous Ensemble models uses multiple models, where all models are different.

Heterogenous Ensemble model techniques:-

A. Voting 
Can be used for both, regression and classification

-> Multiple models like DT, SVC, Log reg., are trained on the same dataset
-> These diff models gives diff predictions like P1,P2,P3,etc
-> On these predictions, the final result is evaluated on the basis of problem:
    If classification = then majority vote (mode) is taken,
    If regression = then mean/avg is taken

B. Stacking

-> Multiple models like DT, SVC, Log reg., are trained on the same dataset
-> These diff models gives diff predictions like P1,P2,P3,etc
-> These predictions data is being passed to a Meta-Model which acts as a final judge and 
   give us a final prediction
