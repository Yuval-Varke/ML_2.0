# Convolutional Neural Network:

* CNN is a neural network mainly used for images and spatial data
* It uses filters (kernels) to scan the input and detect features like edges and patterns
* After convolution, an activation function (like ReLU) introduces non-linearity
* Pooling reduces the size of feature maps while keeping important information
* The output is flattened and passed to fully connected layers for final prediction
* It learns features automatically from simple (edges) to complex (objects)

How Computer Vision works ?
-> Computer vision enables machines to understand images/videos by converting them into numerical data 
  (pixels → arrays)
-> It processes this data using algorithms or neural networks (especially CNNs) to extract features like 
  edges, shapes, textures
-> Low-level features combine into higher-level patterns (objects, faces, scenes)
-> Models are trained on large datasets to learn patterns and make predictions
-> During inference, the trained model analyzes new images and identifies or classifies what it sees
-> Used in tasks like image classification, object detection, segmentation, and recognition

1. Convolutional Layer
-> Min–max scaling (preprocessing) : Convert pixel values to a smaller range (usually 0 to 1) so training is stable
-> Filter (kernel) : A small matrix (like 2×2 or 3×3) that moves across the image and detects patterns
-> Padding (types) : Valid: no padding → output becomes smaller
                  Same: add zeros around edges → output size stays same
-> Feature map (output) : At each position, multiply filter values with the image region and add them → gives a new 
                       matrix highlighting features
-> ReLU activation : Replace all negative values with 0, keep positive values as they are

Simple example:---
Input:
1  2  0
0  1  3
2  2  1

Kernel:
1  0
0 -1

After applying kernel → feature map:
0  -1
1   0

After ReLU:
0  0
1  0

-----------------------------------------------------------------------------------


2. Pooling Layer
-> A pooling layer is used for downsampling, meaning it reduces the spatial size (height and width) of feature 
   maps while keeping the most important information.
It works by dividing the input into small regions (like 2×2) and summarizing each region into a single value.
-> Purpose : Reduces computation and memory usage
          Helps control overfitting
          Keeps important features while discarding less useful details

Types of Pooling:-
-> Max Pooling : Selects the largest value from each region.
                 Most commonly used because it keeps the strongest features.
-> Average Pooling : Takes the average of values in each region.
                     Produces smoother feature maps but may lose sharp features.
-> Min Pooling : Selects the smallest value.
                 Rarely used in practice.

Example (Max Pooling with 2×2 window)
Input:
1  3  2  1
4  6  5  2
7  2  8  3
1  2  0  4

Step: Take max from each 2×2 block

Output:
6  5
7  8

-----------------------------------------------------------------------------------

3. Fully Connected Layer
-> After convolution and pooling, the feature maps are flattened into a single long vector (matrix → 1D array)
-> This vector is passed to a fully connected (dense) neural network, where every neuron is connected to all inputs
-> The dense layers learn high-level combinations of features to make decisions
-> Final layer produces the classification output (e.g., class probabilities using Softmax)

Flow : Feature Maps → Flatten → Fully Connected Layers → Output (Class Prediction)
