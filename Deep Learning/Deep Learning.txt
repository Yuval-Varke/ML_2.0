# Perceptron:- Very basic neuron

* A perceptron is a single neuron model that takes inputs, assigns weights, and adds a bias
* It multiplies inputs with weights, sums them, and passes the result through an activation function
* The activation function (like a step function) decides the final output (0 or 1)
* During training, it adjusts weights based on errors to improve predictions
* It can only solve simple, linearly separable problems (not complex ones like XOR)

# what is Forward and Backward propogation ?
Forward propagation:-
* Data moves from input → hidden layers → output
* Each layer computes weighted sums + activation
* Produces the final prediction

Backward propagation (backpropagation):-
* Compares prediction with actual answer to get error
* Sends this error backward through the network
* Updates weights using gradients to reduce error

In short
* Forward = make prediction
* Backward = learn from mistakes and adjust weights


## Training a neural network:-
# Loss Functions:
For Regression -- 1. MSE - Easily differentiable && Have only 1 global minima | but it's sensitive to outliers
                  2. MAE - Less sensitive to outliers | but it doesn't have a smooth gradient
                  3. HuberLoss - Combination of MSE[for small errors] & MAE[for large erros] && performs better on outliers
                
For Classification -- 
Cross Entropy (CE) -- a. Binary CE --- Nothing but, it's log loss 
                                   --- Activation function used in hidden layers is RelU and at final o/p layer is Sigmoid 
                      b. Categorical CE --- o/p layers neurons = number of classes
                                        --- Activation function used in hidden layers is RelU and at final o/p layer is SoftMax 

# Weight & Bias Updation:
-> Initialize weights and biases (usually small random values)
-> Forward pass: compute prediction using current weights and biases
-> Compute loss: measure error between predicted and actual output
-> Backward pass: calculate gradients (how much each weight and bias affects the error)
-> Update weights:
                w=w−η⋅∂w/∂L​
-> Update biases:
                b=b−η⋅∂b/∂L​
-> Repeat for many iterations (epochs) until error reduces
