== Transformers:-

🔹 What they are
A neural network architecture for sequence tasks that removes recurrence (no RNN) and relies entirely on attention.

Used in:
        translation
        text generation (LLMs)
        summarization



🔹 Core Idea: Attention : Instead of processing step-by-step, transformers look at all words at once 
and learn which words are important to each other.



🔹 Self-Attention (main concept)
Each word interacts with every other word in the sentence.

For each word:

            Query (Q) → what am I looking for
            Key (K) → what do I contain
            Value (V) → actual information
Attention score:
                Attention(Q,K,V) = softmax(Q * K^T / root(dk)) * V

This decides how much focus to give to other words



🔹 Why better than RNN/LSTM
                No sequential processing → parallel computation (fast)
                Captures long-range dependencies easily
                No vanishing gradient issue like RNN




🔹 Transformer Architecture
Two main parts:
1. Encoder
            Stack of identical layers
            Each layer has:
                            Multi-head self-attention
                            Feed-forward network
2. Decoder
Similar to encoder but:
            Uses masked attention (can’t see future words)
            Has encoder-decoder attention



🔹 Multi-Head Attention
            Instead of one attention, use multiple in parallel
            Each head learns different relationships




🔹 Positional Encoding:-
Since no sequence order is inherent:
            Add position information to input embeddings
            Helps model understand word order
